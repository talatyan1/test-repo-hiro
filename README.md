# Crowd Agent: クラウドソーシング自動応募システム

## 概要
クラウドワークス、ランサーズ、ココナラの案件を巡回し、AIによる判定、提案文生成、および自動応募を行うシステムです。

## 主な機能
- **自動巡回**: OpenSearch（各プラットフォーム）による新着案件の取得
- **AI判定**: OpenAI (GPT-4o等) を用いた案件とのマッチング判定
- **提案文生成**: プロンプトに基づいた高品質な提案文の自動作成
- **自動応募**: Playwright/Selenium を用いた応募フォームの自動入力と送信
- **通知機能**: Slack、Email、および各プラットフォームのチャットへの通知
- **セッション管理**: ブラウザプロファイルおよびステート共有によるログイン状態の維持

## セットアップ手順
1.  **依存関係のインストール**:
    ```bash
    pip install -r requirements.txt
    playwright install chromium
    ```
2.  **環境変数の設定**:
    - `.env` ファイルを作成し、必要なAPIキーやログイン情報を記載してください。
    - `credentials.json` (Google Sheets API) を `src/` 配下に配置してください。
3.  **セッションの初期化**:
    - `refresh_sessions.py` を実行して、各プラットフォームへの初回ログインとステート保存を行ってください。

## 使用方法
- **メイン実行**: `python main.py`
- **定期実行**: `run_crowd_agent.bat` をタスクスケジューラに登録

## 注意事項
- このツールは自動応募を行うため、生成された提案文や応募内容はリポジトリに含まれるステータス管理機能（Google Sheets等）で定期的に確認することを強く推奨します。
- 各プラットフォームの利用規約を遵守して使用してください。
