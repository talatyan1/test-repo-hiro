# crowd_agent

クラウドソーシングサイト（CrowdWorks, Lancers, Coconala）の案件を自動で巡回し、AIによる案件判定、提案文の生成、および応募までをサポートするツールです。

## 主な機能

- **案件巡回**: 指定したキーワードに基づき、複数のクラウドソーシングサイトから新着案件を取得します。
- **AI判定**: 取得した案件の内容をAI（Geminiなど）で解析し、ユーザーのスキルや条件に合致するか判定します。
- **提案文生成**: 案件内容に合わせてパーソナライズされた提案文を自動で生成します。
- **自動応募**: Playwrightを使用し、ブラウザ操作を自動化して案件への応募を行います。
- **セッション管理**: ブラウザのプロファイルを管理し、ログイン状態を維持したまま操作が可能です。

## セットアップ

1. **Pythonのインストール**: Python 3.9以上を推奨します。
2. **依存パッケージのインストール**:
   ```bash
   pip install -r requirements.txt
   ```
3. **ブラウザの準備**:
   - Playwrightのブラウザをインストールします。
   ```bash
   playwright install chromium
   ```
4. **環境設定**:
   - `.env` ファイルを作成し、必要なAPIキーや認証情報を設定してください。
   - `credentials.json` などのGoogle API認証情報が必要な場合は配置してください。

## 使い方

メインスクリプトを実行することで、一連のフローを開始できます。
```bash
python main.py
```

## 注意事項

- 本ツールはブラウザ自動操作を含みます。各プラットフォームの利用規約を遵守して使用してください。
- 実行にはAPIキーやセッション情報が必要です。これらは機密情報として扱い、公開リポジトリにはアップロードしないでください。
